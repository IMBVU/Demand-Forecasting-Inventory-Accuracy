# Inventory Demand Forecasting & Accuracy Evaluation (Project 3)

This project analyzes **inventory demand forecasting accuracy** using the Kaggle *Retail Store Inventory Forecasting Dataset*.

## Business Questions
1. How does demand vary over time?
2. How accurate are existing demand forecasts?
3. Which categories show the highest forecast error?
4. Where does inventory planning risk exist?

## KPI Definitions
- Revenue = Units Sold × Price × (1 − Discount)
- Forecast Error = Actual Units Sold − Forecast
- MAE / RMSE / WMAPE used for evaluation

## Outputs
- `data_processed/daily_kpis.csv`
- `data_processed/forecast_accuracy_by_category.csv`
- Forecast accuracy charts in `figures/`
